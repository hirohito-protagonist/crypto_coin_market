and consume [CryptoCompare](https://www.cryptocompare.com/) API.

## Getting started

#### 1. [Setup Flutter](https://flutter.io/setup/)

#### 2. Clone the repo

```sh
$ git clone https://github.com/hirohito-protagonist/crypto_coin_market.git
$ cd crypto_coin_market/
```

#### 3. Run application
```sh
$ flutter run
```

#### 4. (Optional) Run test
```sh
$ flutter test
```
